Debian or Ubuntu Chinese Language Support

Ubuntu Chinese Language Support 编辑/etc/environment如下: vi /etc/environment 添加如下: LC_ALL="en_US.UTF-8" LANG="zh_CN.UTF-8" UNZIP='-O CP936" ZIPINFO="-O CP936" 重新生成语言设置 sudo locale-gen en_US en_US.UTF-8 zh_CN zh_CN.UTF-8 sudo dpkg-reconfigure locales ## 中文支持 ### vim中文支持 编辑

How to Remove LC_CTYPE Error

apt-get autoremove export LC_ALL=en_US.UTF-8 echo "en_US.UTF-8 UTF-8" >> /etc/locale.gen locale-gen
